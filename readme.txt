# Compagnia di Arcieri — Mini Sito Statico

Sezioni:
- Home (index.html)
- Chi siamo (chi-siamo.html)
- Attività (attivita.html)
- Galleria (galleria.html)

Colori/Font:
- Verde primario #1B5E20, Oro #C7902E, Scuro #263238, Chiaro #ECEFF1
- Font: Montserrat (titoli), Inter (testo)

Personalizzazione rapida:
- Modifica email/telefono/indirizzo in `index.html` e `attivita.html`
- Aggiorna timeline, staff e campi in `chi-siamo.html`
- Sostituisci immagini (usa le tue foto o link esterni)

Pubblicazione su GitHub Pages:
1. Crea un repository pubblico (es. `arcieri-sito`) con questi file nella root.
2. Repo → Settings → Pages → “Branch: main” e “/ (root)” → Save.
3. Attendi 1–2 minuti: il sito sarà su `https://USERNAME.github.io/arcieri-sito/`.

Incorporare in Google Sites:
- In Google Sites: Inserisci → Incorpora → URL → incolla l’URL GitHub Pages della Home o di una pagina specifica. Regola altezza/larghezza del riquadro.

Aggiornamenti:
- Modifica i file su GitHub → Commit → l’aggiornamento è online entro ~1 minuto.
